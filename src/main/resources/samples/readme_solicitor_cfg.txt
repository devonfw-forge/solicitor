Overview of the extracted configuration files
=============================================

+ solicitor_sample_copy.cfg: The configuration file

+ licenses_devon4j.xml: sample (maven style) input file

+ sample_configs/LicenseAssignmentSample.xls - decision table data
+ sample_configs/LicenseNameMappingSample.xls - decision table data
+ sample_configs/MultiLicenseSelectionSample.xls - decision table data
+ sample_configs/LicenseSelectionSample.xls - decision table data
+ sample_configs/LegalPreEvaluationSample.xls - decision table data
+ sample_configs/LegalEvaluationSample.xls - decision table data

+ sample_configs/Solicitor_Output_Template_Sample.vm - Velocity report template
+ sample_configs/Solicitor_Output_Template_Sample.xlsx - XLS report template
+ sample_configs/Solicitor_Diff_Template_Sample.vm - Velocity report template (able to show diffs)

to execute Solicitor: call 'java -jar solicitor.jar -c file:solicitor_sample_copy.cfg'
